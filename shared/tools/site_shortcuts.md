# Stable Local Site URLs

Use one command to always keep fixed URLs for each system.

## Start

```bash
bash shared/tools/site_shortcuts.sh start all
```

Or start one system:

```bash
bash shared/tools/site_shortcuts.sh start ziyada
bash shared/tools/site_shortcuts.sh start ali
```

## Open in browser

```bash
bash shared/tools/site_shortcuts.sh open ziyada
bash shared/tools/site_shortcuts.sh open ali
```

## Status

```bash
bash shared/tools/site_shortcuts.sh status
```

## Stop

```bash
bash shared/tools/site_shortcuts.sh stop all
```

## Fixed URLs

- Ziyada Home: http://127.0.0.1:8110/index.html
- Ziyada Arabic: http://127.0.0.1:8110/ar.html
- Ali Home: http://127.0.0.1:8111/index.html
