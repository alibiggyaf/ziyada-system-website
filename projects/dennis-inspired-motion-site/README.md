# Dennis-Inspired Motion Site

Purpose: build an inspired motion/interaction portfolio in a separate project folder without changing existing Ali project files.

## Pages
- `index.html` - home
- `work.html` - project listing
- `work_case_twice.html` - case page with 3D mockup interaction
- `work_case_damai.html` - secondary case page
- `about.html` - about page
- `contact.html` - contact page

## Run
Open `index.html` directly in VS Code browser or via a local static server.

## Notes
- This project is intentionally isolated under `projects/dennis-inspired-motion-site/`.
- Existing files in workspace root remain untouched.
