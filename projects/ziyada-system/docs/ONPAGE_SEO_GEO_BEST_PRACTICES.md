# On-Page SEO & GEO Best Practices for All Pages

Apply these to every important page (e.g., /Home, /Contact, /About):

## 1. Unique, Descriptive Titles
- Use a clear, keyword-rich `<title>` tag for each page.
- Example: `<title>تواصل معنا | زيادة سيستم | Ziyada Systems Contact</title>`

## 2. Meta Descriptions
- Write a unique meta description for each page (150–160 chars).
- Example: `<meta name="description" content="تواصل مع فريق زيادة سيستم لحلول النمو الرقمي المتكاملة. Contact Ziyada Systems for digital growth solutions.">`

## 3. H1 and Heading Structure
- Each page should have one H1 that matches the main topic.
- Use H2/H3 for sections.

## 4. Canonical Tags
- Add `<link rel="canonical" href="https://ziyadasystem.com/Contact" />` to avoid duplicate content issues.

## 5. Internal Linking
- Link between pages using descriptive anchor text (e.g., "Contact us" → /Contact).

## 6. Multilingual Support
- Use `hreflang` tags if you have Arabic/English versions.

## 7. Fast Loading & Mobile Friendly
- Ensure all pages are responsive and load quickly.

## 8. GEO (Generative Engine Optimization)
- Use clear, factual language (not generic AI text).
- Highlight brand, expertise, and unique value.
- Add organization schema (JSON-LD) with brand info.
- Mention "Ziyada System" and core values (e.g., الشراكة) naturally.
- Include social links and real team info.

## 9. Avoid AI-Generated Footprints
- Do NOT use phrases like "As an AI language model..."
- Vary sentence structure and use human examples.
- Add real testimonials, case studies, and media.

---

Apply these to every main page for best SEO and GEO results.
