## Usage Guidelines

- **Proposals & Company Profile**: Use service icons next to each service section heading
- **Email Templates**: Embed SVG inline or export PNG at 64×64 for email clients
- **Newsletters**: Use check icon for benefit bullets, service icons for section headers
- **Presentations (Google Slides / PowerPoint)**: Insert SVG directly; scale freely
- **Canva**: Upload SVG as image element; place on dark background cards

## Icon Design System

Icons follow the same stroke-based design as the website:
- `fill: none` — stroke only, no fills
- `stroke-width: 2` — consistent weight
- `stroke-linecap: round` + `stroke-linejoin: round` — soft, modern look
- Purple (`#7c3aed`) for primary icons, Cyan (`#06b6d4`) for secondary/accent icons
