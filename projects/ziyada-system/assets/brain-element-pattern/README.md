## Usage notes

- SVG is best for presentations and scaling.
- PNG is best for quick drop-in on slides or social graphics.
- All files are transparent and can be placed over dark or gradient backgrounds.
- For layered compositions, place `brain-element-ring-*` and `brain-element-core-*` as separate layers.
- Pattern files can be used as full backgrounds or cropped for headers, covers, and social cards.
