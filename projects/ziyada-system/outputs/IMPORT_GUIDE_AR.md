# 📋 دليل استيراد دليل المبيعات إلى Google Docs

## الطريقة السريعة (5 دقايق)

### الخطوة 1️⃣: أنشئ Google Doc جديد
1. اذهب إلى [docs.google.com](https://docs.google.com)
2. اضغط **"مستند جديد"** (أو ⊕)
3. سماه: **"دليل المبيعات | زيادة سيستم"**

### الخطوة 2️⃣: استورد المحتوى

**الخيار أ: النسخ العادية (الأسهل)**
1. افتح الملف: `SALES_GUIDE_MARKDOWN.md` أو `sales_guide_presentation.html`
2. انسخ كل المحتوى (Ctrl+A ثم Ctrl+C)
3. الصق في Google Doc (Ctrl+V)
4. ارتب التنسيق حسب الاختيار أدناه

**الخيار ب: استيراد HTML مباشرة**
1. في Google Docs: **File → Upload → Select a file**
2. اختر: `sales_guide_presentation.html`
3. Google سيتولى التنسيق تلقائياً
4. ارجع وصحح أي تنسيق زي الألوان والخطوط

### الخطوة 3️⃣: طبق التنسيق (مطابقة الموقع)

**الألوان:**
- الأزرق الأساسي: `#2563eb` (المساعدات والعناوين)
- الأزرق الفاتح: `#3b82f6` (الحدود والتفاصيل)
- النص الأسود: `#0f172a`
- الخلفية الفاتحة: `#f8fafc`

**الخطوط:**
- العنوان الرئيسي: **48pt, Bold**
- العناوين الفرعية: **28pt, Bold** بلون أزرق
- النص العادي: **16pt, Regular**
- الخطوط الموصى بها:
  - Noto Kufi Arabic (عربي)
  - Segoe UI أو Google Sans (إنجليزي)

**التباعد:**
- الهوامش: 0.75 بوصة من كل جانب
- الفواصل بين الأقسام: 20-30 نقطة

## تلميحات ذهبية:

✨ **اطبع PDF من Google Docs لاستخدام بدون إنترنت:**
- File → Download → PDF

✨ **أرسل كـ Word أو PDF:**
- File → Download → Word Document أو PDF

✨ **استخدم "Suggest" Mode للمراجعين:**
- في Google Docs، اضغط وضع **"Suggesting"** لتتبع التغييرات
