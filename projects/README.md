# Projects Index

Each project has an isolated folder under `projects/`.

## Active Projects

- `ziyada-system` - Ziyada automation, content, and application work.

## Standard Project Layout

Each project should include:
- `scripts/`
- `tools/`
- `workflows/`
- `docs/`
- `assets/`
- `notebooks/`
- `outputs/`
- `.tmp/`

## Rule

Do not place project implementation files in workspace root.
